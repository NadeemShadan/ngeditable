# ngEditable
A simple directive to make HTML elements editable in angularjs

NPM
```bash
$ npm i ngeditable
```
BOWER

```bash
$ bower i ngeditable
```

## How to use
Inject in app.js 
```js
angular
    .module('myApp',['ngEditable']);
```
add the directive to the element you want to edit

```html
<div ng-editable>
```
